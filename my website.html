üêß Linux Basics

Linux is a free and open-source operating system used on servers, desktops, and even Android phones. Everything starts from the root directory /, and common folders include /home (user files), /bin (commands), and /etc (configurations).

Useful commands: ls (list files), cd (change folder), cp (copy), mv (move/rename), rm (delete), mkdir (make folder).

With sudo you run admin commands. Permissions control who can read, write, or run files. For networking, ping tests connections and ssh lets you connect to remote machines.

In short: Linux is powerful, stable, and flexible ‚Äî mastering its commands gives you control over your system.
